Welcome to GitHub Pages.

This automatic page generator is the easiest way to create beautiful pages for all of your projects. Author your page content here using GitHub Flavored Markdown, select a template crafted by a designer, and publish. After your page is generated, you can check out the new gh-pages branch locally. If you’re using GitHub Desktop, simply sync your repository and you’ll see the new branch.

Designer Templates

We’ve crafted some handsome templates for you to use. Go ahead and click 'Continue to layouts' to browse through them. You can easily go back to edit your page before publishing. After publishing your page, you can revisit the page generator and switch to another theme. Your Page content will be preserved.

Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.

至于台湾人民怎么想？

我想最令人关注的应该是台湾新生代年轻人对大陆的排斥。因为看来我们这一代的人是看不到两岸统一了，而三十年，五十年后，我们化为尘土，后一代，甚至二代，三代之后的人怎么想？ 这才是重点。 而目前台湾愈年青的人独性愈强，（有人称他们为「天然独」），这才是未来两岸统一最大的障碍，而年轻人反统的原因可就复杂了。 有人说，是否李登辉，阿扁推行本土教育的结果，这点我祗同意一半，因为日本人统治台湾50年，严厉推行「皇民教育」，但二次世界大战后，台湾光复回归中国，台湾人民还是万民空巷的游行庆祝回归祖国，若不是陈仪的无能，最后演变成 二二八事变，两岸人民融合相处应该不会是问题。 造成年青一代排斥大陆最主要是社会的多元化，共产党的政策向来是拉一个打一个，他们认为拉台商打台独，最后可以达到以商领政的目的。 可惜投资大陆的台商赚了钱回台湾炒房，让年轻人买不起房，而且让台湾贫富悬殊更加严重，加以在民主选举政治中，商人向来是最被排斥的一群，其结果是反商变成了反共。 蔡英文就职后的两岸关系- 黄若

第二，年青人本来就充满叛逆性，这和共产党的「威权」政治注定是格格不入。「您凭什么要管我要不要统一？」国家民族大义？伟大的中国梦？台湾青年教育中没有这一课。（说真的西方教育也没有）台湾青年要的是与他们切身有关的小确幸，有工作，有加薪，有房，有车…不实际的未来梦对他们是没有吸引力的。

第三，世代隔绝，台湾现在支持统一的基本上都是1949年前后从大陆撤退的蒋家政权子民和他们的第二代（第三代就不一定了），两岸隔阂愈久，疏离感愈大，统一愈困难，这是自然定律，谁也扭转不了。

有一位新华社的朋友问我：没有九二共识的两岸关系如何走下去？ 我说，中共一直强调九二共识，但从来没有把九二共识的来龙去脉讲清楚。这个问题要从九二共识的起源谈起.

九二年，两岸在香港会谈，达成了两岸同属一个中国的共识，但对于一中的内涵，两岸各有说法，中共认为当然是「中华人民共和国」，但台湾则坚持是 「中华民国」，这是「一中各表」的由来，但由于当时中共不肯同意签署「一中各表」的文件，（所以叫共识，而不叫备忘录，或协议）所以祗是口头的约定。

等民进党上台后，在移交清单中并没有找到双方签定的文字文件，加以民进党本来就反对 「一中」因此要叫民进党去接受就变得困难重重了。

所以蔡英文在演讲中说：「承认1992年两岸两会会谈的历史事实，与求同存异的共同认知，这是历史事实。」我想这已是蔡英文和民进党的让步，也是台湾人民共同认定的底线，没有签署正式文件，坦白说，中共要承担更多的责任。

蔡英文就职后的两岸关系- 黄若 没有九二共识，两岸关系就不能继续往前走吗？当然不是，国民党的九二共识没有办法为人民接受，最后败选的最主要原因是，马英九在接受中共的九二共识时，祗是一味的退让，并没有为台湾人民争取到最佳利益. 例如：国际空间，两岸和平协议…，如果蔡英文接受了马英九的「九二共识」，台湾接下去已没有回旋空间了，退回到九二共识之前，重谈两岸关系是蔡英文的手段，但中共是不会同意的，好不容易框你在笼子里，怎么可能再退回去重启谈判？ 蔡英文是谈判高手，台湾 WTO 的主谈者，她当然也暸解这博弈中的风险。其实以第三者的眼光来看，中共若能和她谈判并取得一个结果，相信比马英九的谈判结果更能得到台湾人民的信服，祗因为马不是在台湾出生，台湾人民不认为他在两岸关系上的处理方式代表台湾人民。 所以中共面临的是重启谈判？签订新合约？还是反目相向？坦白说球在中方，不在台方。中国有的是资源可以对付台湾，例如武力制裁台湾，也可以拉拢台湾邦交国家和台湾断交，甚至降低两岸贸易，减少两岸之间人民旅游往来，但任何动作都难免会伤害到台湾人民的感情，让两岸国际愈走愈远，所以中共的困难点变得是「投鼠忌器」了。 其实和中国大陆一样，蔡英文目前所面临的是国内的经济问题，而不是两岸问题，台湾的经济增长已低到 1% 都不到，甚至有可能负增长。 因此蔡英文的执政重心会放在内政而不是两岸关系，对两岸关系蔡英文采取「以拖待变」的政策，所以中共的善意反应当然很好，但如果是激烈的反制行为相信蔡英文也不会太意外的。
